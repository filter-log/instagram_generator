# Instagram Feed Generator

GitHub Pages에서 바로 실행할 수 있는 정적 버전입니다. 브라우저에서 이미지를 합성하고 결과물을 ZIP으로 다운로드합니다.

## 배포

1. GitHub 저장소의 `Settings > Pages`로 이동합니다.
2. `Build and deployment`에서 `Deploy from a branch`를 선택합니다.
3. `Branch`를 `main`, 폴더를 `/ (root)`로 선택합니다.

## 지원 형식

- 입력: 브라우저가 읽을 수 있는 PNG, JPG, WebP, GIF, AVIF
- 출력: JPG 또는 PNG

PSD 처리는 서버 없는 GitHub Pages 환경에서 지원하지 않습니다. PSD 원본은 PNG 또는 JPG로 변환한 뒤 업로드하세요.
